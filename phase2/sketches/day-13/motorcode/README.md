# Motor and Valve Control System with Emergency Stop

## Overview

This project implements a motor and valve control system using an Arduino Uno. The system allows a user to control motor speed through serial commands, operate a relay-controlled valve, and monitor system status using an LED indicator.

A hardware Emergency Stop (E-Stop) button is implemented using interrupts to immediately stop all operations and place the system in a safe state.

## Features

* PWM-based DC motor speed control
* Relay-controlled valve operation
* Serial command interface
* LED status indication
* Interrupt-driven Emergency Stop
* Manual system reset after emergency events
* Safe shutdown of motor and valve

## System Operation

### Motor Control

The motor speed is controlled using PWM values generated by the Arduino.

Serial Commands:

| Command | Action          |
| ------- | --------------- |
| 1 - 9   | Set motor speed |
| 0       | Stop motor      |

Speed levels are mapped to PWM values ranging from 80 to 255.

### Valve Control

| Command | Action      |
| ------- | ----------- |
| o       | Open Valve  |
| f       | Close Valve |

The valve is controlled through a relay module connected to pin D6.

### LED Indicator

* LED blinks every 500 ms while the motor is running.
* LED remains OFF when the motor is stopped.

---

## Emergency Stop System

An interrupt is attached to the Emergency Stop button.

When activated:

* Motor immediately stops.
* Relay is switched OFF.
* Valve closes.
* LED turns OFF.
* System enters Emergency Mode.

Message displayed:

```text
EMERGENCY STOP ACTIVATED
Send 'r' to reset system
```

### Reset Command

```text
r
```

This clears the emergency condition and returns the system to normal operation.
