\# Automated Grow-Bench Prototype



\## Overview



The Automated Grow-Bench Prototype is an Arduino Uno-based smart irrigation system developed to automate watering using soil moisture measurements. The system continuously monitors the moisture level of the soil through a soil moisture sensor connected to the analog input pin. When the soil becomes dry, the relay module activates the water pump, and irrigation stops when the moisture level reaches the desired threshold.



The prototype supports both automatic and manual modes of operation. A push button is used to enable manual override, while serial commands (`STATUS`, `FORCE\_ON`, and `FORCE\_OFF`) allow monitoring and control through the Serial Monitor. The system provides a simple and reliable solution for efficient water management and demonstrates the practical application of embedded systems in smart agriculture.
